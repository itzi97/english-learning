A personal space to track vocabulary and books as I improve my English.

## Contents

- **[vocabulary.md](vocabulary.md)** — Words I've encountered and had trouble understanding, with definitions and examples.
- **[books/reading-list.md](books/reading-list.md)** — Books I want to read.
